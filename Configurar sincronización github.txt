/** CONFIGURAR SINCRONIZACIÓN CON GITHUB **/

Creamos un repositorio en GitHub, copiamos la url y ejecutamos el comando:

	git clone https://github.com/raulcb98/RC1819.git

La primera vez que trabajemos con git tenemos que configurar nuestro email y nombre de usuario de la cuenta git en nuestro equipo con los siguientes comandos:

	git config --global user.email "raul.castilla215@alu.uhu.es"
	git config --global user.name "raulcb98"

Para subir archivos a nuestro repositorio tenemos que ejecutar los siguientes tres comandos:

	git add . (añade todos los ficheros que haya creado nuevos)
	git commit -a -m "Primer programa Prolog" (Envía un mensaje)
	git push origin master

Para descargar los cambios tenemos que hacer un pull por si tenemos una versión desactualizada del repositorio.

	git pull origin master
